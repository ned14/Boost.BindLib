Detailed Description

template<class HeadBaseType, class ItemType>
class quickcpplib::_xxx::algorithm::bitwise_trie::bitwise_trie_head_accessors< HeadBaseType, ItemType >

Default accessor for a bitwise trie index head.

Template Parameters
HeadBaseTypeThe type from which bitwise_trie inherits
ItemTypeThe type of item indexed.

This default accessor requires the following member variables in the trie index head type:

  • <unsigned type> trie_count
  • ItemType *trie_children[8 * sizeof(<unsigned type>)]
  • bool trie_nobbledir (if you use equal nobbling only)
